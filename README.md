# TG Click - Система создания интернет-магазинов

Веб-приложение для создания и управления интернет-магазинами с различными шаблонами.

## Функциональность

### Авторизация
- Вход в систему с логином и паролем
- Логин: `ritter`
- Пароль: `Vbh22tcnm`

### Главное меню
- Создание нового магазина
- Просмотр списка магазинов

### Шаблоны магазинов
- **Мобильные телефоны** - шаблон для магазина мобильных устройств
- **Цветы** - шаблон для цветочного магазина
- **Одежда** - шаблон для магазина одежды
- **Игрушки** - шаблон для магазина игрушек
- **Мебель** - шаблон для мебельного магазина

### Магазин цветов (реализован)
- Адаптивный дизайн (2 карточки на мобильных, 4 на десктопе)
- Рекламный баннер с возможностью редактирования
- Навигация по категориям
- Добавление товаров с загрузкой изображений
- Редактирование товаров
- Редактирование баннера с загрузкой фонового изображения
- Сохранение данных в localStorage

## Технологии

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage для хранения данных

## Запуск

1. Откройте `index.html` в браузере
2. Или запустите локальный сервер:
   ```bash
   python3 -m http.server 8000
   ```
3. Откройте в браузере: `http://localhost:8000`
